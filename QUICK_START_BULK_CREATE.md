## 🎯 How to Use Bulk Create in 3 Steps

### Step 1️⃣: Click the Bulk Create Button
Navigate to your project's **Tasks** page and click the **"🚀 Bulk Create"** button in the header.

```
┌─────────────────────────────────────────┐
│  Tasks Page Header                      │
│  ┌─────────┬──────────────┬──────────┐ │
│  │ ⚡Quick │ 🚀 BULK      │ 📋 Full  │ │
│  │  Add   │ CREATE ← Click here! │ Create │ │
│  └─────────┴──────────────┴──────────┘ │
└─────────────────────────────────────────┘
```

### Step 2️⃣: Paste Your Task List
A dialog opens. Copy & paste your task names (one per line):

```
┌──────────────────────────────────────────┐
│ 🚀 Bulk Create Tasks                    │
│ ┌──────────────────────────────────────┐│
│ │ Task Names (one per line)      (5)  ││
│ ├──────────────────────────────────────┤│
│ │ Fix login page loading issue         ││
│ │ Add dark mode support               ││
│ │ Update API documentation            ││
│ │ Implement caching layer             ││
│ │ Optimize database queries           ││
│ │                                      ││
│ └──────────────────────────────────────┘│
│                                          │
│ Type:      [TASK ▼]                     │
│ Priority:  [P2 ▼]                       │
│ Assignee:  [Unassigned ▼]               │
│                                          │
│ Summary: 5 tasks to create               │
│                                          │
│  [Cancel]  [Create 5 Tasks] ───→ CLICK! │
└──────────────────────────────────────────┘
```

### Step 3️⃣: Set Config & Click Create
- Select **Type** (applies to all tasks)
- Select **Priority** (applies to all tasks)
- Select **Assignee** (applies to all tasks)
- Click **"Create X Tasks"**

Watch the progress bar fill up! ✨

## ✨ Features of Bulk Create

| Feature | How It Works |
|---------|-------------|
| **🔢 Task Counting** | Auto-counts how many tasks you'll create |
| **📊 Progress Bar** | See real-time progress as tasks are created |
| **⚠️ Error Handling** | Tells you exactly which tasks succeeded/failed |

| **✂️ Auto-Trim** | Leading/trailing spaces removed automatically |
| **🔄 Same Settings** | Type/Priority/Assignee applies to ALL tasks |

## 💡 Pro Tips

### Tip 1: Copy from Spreadsheet
1. Open your Excel/Google Sheet
2. Select the task name column
3. Copy (Ctrl+C)
4. Open Bulk Create dialog
5. Paste (Ctrl+V)
6. Done! ✨

### Tip 2: Format Consistently
Good formatting for bulk create:
```
Fix typo in header
Implement caching
Update documentation
```

Bad formatting (avoid):
```
FIX TYPO IN HEADER!!!
implement CACHING [urgent]
UPDATE docs - ASAP!!!
```

**Keep it simple: just the task name!**

### Tip 3: Create Unassigned First
1. Set Assignee to "Unassigned"
2. Create all tasks
3. Then bulk assign by filtering and selecting

This is faster than trying to assign individually while creating.

### Tip 4: Use Default Priority
1. Set Priority to P2 or P3 (safe default)
2. Create all tasks
3. Adjust outliers individually later

This saves time vs. trying to remember each task's priority.

### Tip 5: Group by Type
Create all bugs together, then all features:
```
First bulk create (Type: BUG):
- Login page loading slow
- Dashboard chart wrong
- Export PDF broken

Then bulk create (Type: FEATURE):
- Add dark mode
- Implement 2FA
- Create mobile app
```

## ❓ Frequently Asked Questions

**Q: Can I use Bulk Create to create 100 tasks?**  
A: Yes! No limit. Though you might want to split into multiple bulk creates for organization.

**Q: What if a task name is too long?**  
A: It's fine! Task names can be as long as needed.

**Q: Can I set different priorities for each task in bulk create?**  
A: Not directly. Create them all with one priority, then bulk edit to adjust.

**Q: Are my bulk create settings saved?**  
A: Yes! Your last Type, Priority, and Assignee are automatically saved.

**Q: What if some tasks fail to create?**  
A: You get immediate feedback. The dialog tells you which ones succeeded and which failed.

**Q: Can I add descriptions while bulk creating?**  
A: No, bulk create is for names only. Use Full Create for tasks needing descriptions.

**Q: Can I set deadlines with bulk create?**  
A: No, use Full Create for deadlines. You can add them after bulk creation if needed.

**Q: Do I lose the tasks if creation fails?**  
A: Not at all! Bulk create creates tasks one by one. If one fails, the others are already created.
